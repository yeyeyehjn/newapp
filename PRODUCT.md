# Product Context

## What is this?

一个小程序管理系统的前端界面，提供案件管理、任务中心、工作台等功能模块。

## Target Users

- 仲裁员

## Core Features

1. **工作台 (Workbench)** - 主工作界面
2. **案件管理** - 查看和管理案件列表、详情
3. **任务中心** - 处理待办任务
4. **数据统计** - 查看案件统计数据
5. **个人中心** - 用户资料管理

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS (assumed based on modern setup)

## Design Goals

- 清晰的信息层级
- 高效的操作流程
- 专业的企业级界面
- 响应式布局

## Current State

项目已有基础组件结构，需要完善 UI/UX 设计和视觉系统。
